# Qt 经典游戏合集 🎮
基于 **Qt C++** 开发的多款超流畅经典小游戏，包含：贪吃蛇、俄罗斯方块、飞机大战...
代码结构清晰、注释完整、可直接编译运行，非常适合学习 Qt 图形编程与游戏开发。

---

## 游戏列表
| 游戏名称 | 核心玩法 | 操作说明 |
|--------|----------|----------|
| **贪吃蛇 SnakeGame** | 控制蛇吃食物，越长难度越高 | ↑ ↓ ← → 控制方向 |
| **俄罗斯方块 TetrisGame** | 经典消除玩法，累积得分 | ↑ 旋转 / ←→ 移动 / ↓ 加速下落 |
| **飞机大战 PlaneGame** | 射击敌机、躲避碰撞、支持多难度 | ↑↓←→ 移动 / 空格射击 / 1 2 3 切换难度 |
| **FlappyBird** | 点击飞越管道，躲避障碍 | 鼠标/空格跳跃 |

## 编译运行
1. 安装 Qt 开发环境（Qt 5 / Qt 6 均可）
2. 使用 Qt Creator 打开任意游戏内的 `CMakeLists.txt` 或 `.pro`
3. 点击 **构建 → 运行**
